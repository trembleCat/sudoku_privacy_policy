1. 我们收集的信息
在您使用本应用的过程中：

我们不收集个人身份信息 (PII)：我们不会收集您的姓名、邮箱、电话号码或任何可识别您个人身份的信息。

游戏数据与 iCloud：为了实现多设备间游戏进度同步，本应用会使用 Apple 的 iCloud 服务。这些数据存储在您个人的 iCloud 账户中，受 Apple 的隐私保护体系管理。我们无法访问、读取或共享您 iCloud 中的任何数据。

2. 第三方服务与广告
无广告 SDK：本应用不包含任何第三方广告 SDK（如 AdMob、Facebook Audience Network 等）。我们不会跟踪您的行为以投放广告。

无数据分析 SDK：我们不使用任何第三方分析工具来收集您的行为数据。

3. 数据安全
由于我们不收集您的个人信息，因此不存在数据泄露的风险。您的游戏数据安全完全由您个人的 iCloud 账户安全性决定。我们建议您设置强密码并开启双重验证以保障您的 iCloud 安全。

本应用面向所有年龄段用户，且不收集任何个人数据，因此无需针对未成年人进行特殊的数据处理。
